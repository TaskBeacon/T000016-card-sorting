Title: Card Sorting Task (WCST-style)
Construct: cognitive flexibility / set shifting / rule maintenance

Purpose:
Show the participant-visible card sorting timeline for color, shape, and number rule blocks.

Rows / conditions:
- Color rule: sort the target card by matching color to one of four reference cards.
- Shape rule: sort the target card by matching shape to one of four reference cards.
- Number rule: sort the target card by matching number to one of four reference cards.

Timeline:
- Rule cue: 400 ms; no response; current rule text.
- Anticipation: 200 ms; no response; fixation +.
- Card choice: 2000 ms; press 1-4; target card above four reference cards.
- Feedback: 600 ms; no response; correct or incorrect.
- ITI: 300 ms; no response; fixation +.

Notes:
- The active rule changes by block in the sequence color -> shape -> number.
- Keys 1-4 map left-to-right to the four reference cards.
- Correct key is computed from the target-reference match under the active rule.
